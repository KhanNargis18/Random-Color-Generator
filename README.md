## Introduction
A JavaScript-based random color generator that changes background color and automatically copies the generated HEX code to clipboard

## Features
- Generates a random background color
- Displays HEX color code
- Automatically copies HEX code to clipboard when button is clicked
- Simple and user-friendly interface
- Fully responsive design

## Technologies Used
- HTML5: A markup language used to structure the web page.
- CSS3: A styling language used to design layout, colors, and responsiveness.
- JavaScript: A programming language used to generate random colors and handle user interactions.
- Git: A version control system for tracking project changes.
- GitHub: A platform for storing, managing, and sharing the project repository.

## Installation
To run this portfolio locally, follow these steps:

1. Clone the repository <br>
git clone https://github.com/KhanNargis18/Random-Color-Generator.git

2. Open project folder <br>
cd random-color-generator

3. Open index.html in browser

## Usage

1. Open the application in your web browser.
2. Click the Generate Color button.
3. The background color will change randomly.
4. The generated HEX color code will be displayed on the screen.
5. The HEX color code will automatically be copied to your clipboard.
6. Paste the copied color code anywhere you want to use it.
